Please click either button above to see the weather data of either your "Home" location (Determined by I.P. Address). Or an "Other" place will provide the weather in a randomly selected city & country from around the world. You can also type the name of a city and if it's found in the database the weather data of that city will be shown. Please note that if the city name is a common name it may select the first reference from the database.

Database data sourced from: Kaggle and running from SQLite3.

This program is using the following Python Packages: Django 5.0, Geocoder, Requests, datetime, and the Mateo Weather Forecast API which references public weather data from DWD ICON, NOAA GFS, MeteoFrance, ECMWF, JMA, MET Norway, and GEM resources to report the temperatures.

Version 1.0 - Home Weather Data properly retrieved from I.P. Address.

Version 1.1 - "Other" button added and now randomly searches the database for the weather in over 44,000 locations by geo-coordinates.
